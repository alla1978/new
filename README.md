Баг репорт 

Название: Некорректная работа кнопки "Назад" при регистрации по номеру телефона

Описание:
При проведении процесса регистрации в мобильном приложении Magnum Go с использованием номера телефона, обнаружена проблема с функциональностью кнопки "Назад".

Шаги воспроизведения:
Откройте приложение Magnum Go.
Введите действующий номер телефона
Нажать кнопку Продолжить
Нажать на кнопку Назад (для возврата на страницу ввода номера телефона)

Ожидаемый результат:
При нажатии на кнопку "Назад" ожидается возврат к предыдущему экрану

Фактический результат:
При нажатии на кнопку "Назад" не происходит возврата на предыдущий экран.

Приоритет: Средний

Устройство: Xiaomi 11Lite 5G NE
Операционная система: Xiaomi MIUI 14 (Android 13)
